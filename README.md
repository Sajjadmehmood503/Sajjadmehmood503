### 🚀 What I do

- Design and implement **REST APIs** and backend modules in **Yii2** and **Laravel**, with attention to validation and data integrity
- Work on **financial / reporting** domains (e.g. chart of accounts, trial balance, P&amp;L, multi-currency concepts) in Yii2-backed services
- Document APIs with **OpenAPI / Swagger** and secure routes with **bearer / JWT-style** patterns where needed
- Integrate services over HTTP (**Guzzle**) — e.g. **file upload flows** to companion microservices (**bm-service**)
- Ship and iterate with **Docker** images and **Kubernetes** manifests for microservice-style deployments
- Support **SQL Server → MySQL** migration workflows using **Python** helpers for dump conversion and cleanup
- Deliver **WordPress** customization (real estate listings), **Yii2 LMS** features (attendance, admin/backend flows), and responsive **business sites** (e.g. [fastmovingnow.com.pk](https://fastmovingnow.com.pk))
